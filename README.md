# Academia OOP 🏋️

> ⚠️ Projeto desenvolvido para fins de estudo e prática de POO em Python.

Sistema de gerenciamento de academia desenvolvido em Python, aplicando os conceitos de Orientação a Objetos.

## 📋 Sobre
Sistema que permite cadastrar alunos e personal trainers, gerenciar matrículas, planos e contratação de personals. Desenvolvido como projeto de aprendizado durante meus estudos de Python.

## 🚀 Tecnologias
- Python 3.12

## 📁 Estrutura do Projeto
```
academia-oop/
├── modelos/
│   ├── aluno.py
│   ├── personal.py
│   ├── pessoa.py
│   └── planos.py
├── sistema/
│   └── academia.py
├── utils/
│   └── cadastro.py
└── main.py
```

## ▶️ Como executar

1. Clone o repositório
```
git clone https://github.com/jheffx/academia-oop.git
```

2. Execute o arquivo principal
```
python main.py
```

## 📚 Conceitos aplicados
- Herança e Polimorfismo
- Encapsulamento com `@property`
- Abstração e Responsabilidade Única
- Dunder methods
- Tratamento de exceções com `try/except`
- Organização em módulos e pacotes
